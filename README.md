# LandsITrack Dashboard

LandsITrack Dashboard is an IoT-based landslide monitoring system that uses Arduino, ESP32, and sensors to detect early warning signs of landslides.

This thesis project provides real-time monitoring of:

* Tilt angles (MPU6050)
* Soil moisture levels
* Vibration detection
* Alert levels for landslide risk and early warning

## System Overview

The system collects environmental data using Arduino and sends it to an ESP32 for processing and cloud upload. A dashboard displays real-time data for monitoring and analysis.

## Technologies Used

* Arduino Uno
* ESP32
* GSM SIM800
* MPU6050 (tilt sensor)
* Soil Moisture Sensors
* ThingSpeak Dashboard
